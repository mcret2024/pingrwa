1단계 · Wind-down 시작
서비스 종료를 공식 공지합니다. announced_at 기록. 이 단계 이후 2~5단계가 활성됩니다.
2단계 · AMM 매수 풀 설정 (선택)
회원이 SilicaSTO 를 USDT 로 매도할 수 있는 자동 매수 풀. 매수 시세는 회사 재량으로 결정 — 1토큰=1USDT 보장 안 함. USDT 유동성 소진 시 매수 중단.
3단계 · 출금 마감일 설정
이 시점 이후 사이트는 영구 폐쇄되고, 미회수 자금은 회사에 귀속됩니다 (약관 제10.5조). announced_at 으로부터 최대 2개월 이내 지정 가능. 시각은 자동으로 23:59:59 KST 적용.
4단계 · 스테이킹 강제 중단
모든 stakers 의 SilicaSTO 가 즉시 언스테이킹되어 보유 잔액으로 이동합니다.
14~16일 KST 락 윈도우 기간에는 실행 불가. 관리자 지갑 연결 필수.
5단계 · 영구 폐쇄
사이트가 영구 폐쇄됩니다. 출금 마감일 이후에 진행하세요. 미회수 자금 합계가 audit 에 기록되고 state 가 closed 로 변경됩니다. 이후 모든 user 측 API 가 차단됩니다.
